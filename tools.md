# CLI Tools

Tools I expect to be installed on my development machines.

## Essential

| Tool | Purpose |
|------|---------|
| `eza` | Modern `ls` replacement with icons and colors |
| `git` | Version control |
| `vim` | Terminal editor |
| `tmux` | Terminal multiplexer |
| `conda` | Python environment management (miniconda3) |
| `pnpm` | Node.js package manager |
| `htop` | Interactive process/memory/CPU monitor |

## Shell Plugins

| Feature | Plugin |
|---------|--------|
| Inline suggestions (fish-style) | `zsh-autosuggestions` |
| Syntax highlighting | `zsh-syntax-highlighting` |

## Nice to Have

| Tool | Purpose |
|------|---------|
| `gh` | GitHub CLI for PRs, issues, etc. |
| `jq` | JSON processor |
| `ripgrep` (`rg`) | Fast code search |
| `fd` | Modern `find` replacement |
| `bat` | `cat` with syntax highlighting |
| `docker` | Container runtime |
