# SSH Preferences

## Hosts

Use short aliases for frequently accessed hosts. Add them here as needed. Example format:

| Alias | Host | User | Key | Purpose |
|-------|------|------|-----|---------|
| `myserver` | 1.2.3.4 | myuser | `~/.ssh/id_rsa` | Example server |

Short aliases allow `ssh myserver` instead of typing the full connection string.
